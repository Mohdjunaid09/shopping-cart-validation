# 🛒 Shopping Cart Validation Project

## 📌 Project Overview

This project demonstrates **client-side validation using JavaScript** for a basic shopping cart system. It includes three main modules:

* 🔐 Registration Page
* 🔑 Login Page
* 🛍️ Cart Page

The validation ensures that users enter correct and complete data before submitting forms.

---

## 🚀 Features

* ✅ Registration form validation

  * Checks empty fields
  * Password length validation
  * Confirm password match

* ✅ Login form validation

  * Ensures all fields are filled

* ✅ Cart form validation

  * Validates product name
  * Ensures quantity is greater than 0

---

## 🛠️ Technologies Used

* HTML5

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Right-click on `register.html`
4. Select **Open with Live Server**

---

## 💡 How Validation Works

* JavaScript is used to validate user input
* `onsubmit` event prevents form submission if validation fails
* Error messages are displayed dynamically

---

## 📷 Output

* Registration success alert
* Login success alert
* Cart item added alert
